# ZYF Auto-Crop - ComfyUI 人物智能裁剪插件

ComfyUI 的智能人物裁剪插件，支持自动检测人脸和身体，智能裁剪人物主体，并提供丰富的裁剪和缩放选项。

## 功能特点

### 1. ZYF(按人物主体)比例缩放节点
- **智能人物检测**：自动检测人脸和身体，采用"人脸优先、身体次之"的裁剪策略
- **自动宽高比**：开启后根据输入图像的横竖方向自动反转比例（如选 2:3 遇横图自动变 3:2）
- **灵活的宽高比**：支持多种预设宽高比（原始、1:1、3:2、4:3、16:9、2:3、3:4、9:16）或自定义比例
- **精确的尺寸控制**：
  - 缩放到边：支持最长边、最短边、宽度、高度、总像素(千像素)
  - 倍数取整：向上取整到指定倍数（8/16/32/64/128/256/512/无）
- **高质量缩放**：支持多种采样方法（lanczos、nearest、bilinear、bicubic）
- **多人物支持**：可选择裁剪第几个人物（0=所有人合成区域，1=最大人物，以此类推）
- **输出端口**：
  - 图像：裁剪后的图像
  - 遮罩：对应的遮罩

### 2. ZYF(按人物主体)比例缩放(2)节点
- 继承"ZYF(按人物主体)比例缩放"节点的全部功能
- **设备选择**：支持 `cpu/cuda`，默认 `cpu`，降低显存占用并在推理后自动释放显存
- **模型选择与自动下载**：可选择人脸/人物模型，缺失时自动从 `hf-mirror.com` 或 `huggingface.co` 下载到 ComfyUI 目录

### 3. ZYF(聚焦人体裁剪)节点
- **智能头部/人脸/身体检测**：使用 YOLOv8 模型精准识别头部（包含整个头部+头发）、人脸和人体区域
- **多种裁剪范围**：头像特写 / 胸部以上 / 腰部以上 / 臀部以上 / 膝盖以上 / 全身照
- **智能裁剪比例**：0.0=自动（使用裁剪范围预设），0.4-0.5=全身照，0.6-0.7=半身照，0.8-0.9=头像特写
- **尽量保持原图比例**：高度由裁剪范围决定，宽度按原图宽高比计算，裁剪掉所选范围周边多余的部分
- **自动宽高比**：开启后根据输入图像的横竖方向自动反转比例
- **灵活的宽高比**：选择比例时对裁剪范围后的图片进行比例裁剪（缩小），尽量不把人脸裁剪掉
- **精确的尺寸控制**：缩放到边、倍数取整、多种采样方法
- **多人物支持**：可选择裁剪第几个人物（按面积排序）
- **可调边距**：支持设置头部周围的额外边距（像素）
- **容错处理**：头部检测失败时自动回退到人脸检测；未检测到头部/人脸时返回原图
- **输出端口**：
  - 图像：裁剪后的图像
  - 遮罩：对应的遮罩

### 4. ZYF(聚焦人体裁剪)预览节点
- 生成裁剪区域可视化预览，帮助用户调整参数
- 在图像上绘制裁剪框（红色）和检测框（绿色=头部，黄色=人脸，紫色=身体）
- 输出裁剪信息文本（裁剪范围、比例、尺寸、检测结果）
- **输出端口**：
  - 预览图像：带标注的预览图像
  - 裁剪信息：裁剪参数和检测信息的文本

### 5. ZYF自动裁剪边框节点
- **智能边框检测**：自动检测并裁剪图像四周的纯色边框
- **可调容差**：支持设置颜色容差值，灵活控制裁剪范围
- **安全裁剪**：设置最小裁剪尺寸，防止过度裁剪
- **输出端口**：
  - 图像：裁剪后的图像
  - 遮罩：对应的遮罩

## 安装方法

1. 将本插件文件夹放入 ComfyUI 的 `custom_nodes` 目录
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 模型会在首次使用时自动下载（无需手动下载）

## 使用的模型

本插件使用以下 YOLOv8 模型：

### 1. 人脸检测模型
- **文件名**: `face_yolov8m.pt`（可选 `face_yolov8n.pt`、`face_yolov8n_v2.pt`、`face_yolov8s.pt`）
- **本地路径**: `ComfyUI/models/ultralytics/bbox/face_yolov8m.pt`
- **来源**: [ashllay/YOLO_Models](https://huggingface.co/ashllay/YOLO_Models)
- **自动下载**: 优先从 hf-mirror.com 下载，失败则从 huggingface.co 下载

### 2. 头部检测模型
- **文件名**: `head_yolov8n.pt`
- **本地路径**: `ComfyUI/models/ultralytics/bbox/head_yolov8n.pt`
- **来源**: [jadechip/head_yolov8n.pt](https://huggingface.co/jadechip/head_yolov8n.pt)
- **自动下载**: 优先从 hf-mirror.com 下载，失败则从 huggingface.co 下载

### 3. 身体检测模型
- **文件名**: `person_yolov8m-seg.pt`（可选 `person_yolov8n-seg.pt`、`person_yolov8s-seg.pt`）
- **本地路径**: `ComfyUI/models/ultralytics/segm/person_yolov8m-seg.pt`
- **来源**: [ashllay/YOLO_Models](https://huggingface.co/ashllay/YOLO_Models)
- **自动下载**: 优先从 hf-mirror.com 下载，失败则从 huggingface.co 下载

**注意**: 
- 首次使用时，插件会自动检测模型是否存在
- 如果模型不存在，会自动从 Hugging Face 下载（优先使用国内镜像）
- 下载过程会显示进度信息
- 如果自动下载失败，可以手动下载模型并放置到对应路径

## 使用说明

### ZYF(按人物主体)比例缩放节点

**输入参数：**
- **图像**：输入图像
- **自动宽高比**：开启后根据图像横竖方向自动反转非"原始"的比例（默认关闭）
- **遮罩**（可选）：输入遮罩
- **宽高比**：目标宽高比选择（默认"原始"）
- **比例宽/高**：自定义宽高比的值（仅当选择"自定义"时生效）
- **缩放模式**：适应/裁剪/填充（默认"适应"）
- **采样方法**：lanczos/nearest/bilinear/bicubic（默认"lanczos"）
- **倍数取整**：8/16/32/64/128/256/512/无（默认"16"）
- **缩放到边**：无/最长边/最短边/宽度/高度/总像素(千像素)（默认"无"）
- **缩放到长度**：目标边的像素长度（默认1024）
- **背景颜色**：十六进制颜色代码（默认"#000000"）
- **人物索引**：选择第几个人物（默认0=所有人合成区域，1=最大人物，以此类推）

**输出：**
- 图像：裁剪后的图像
- 遮罩：对应的遮罩

**裁剪策略：**
1. 检测人脸和身体
2. 最大化显示人物主体部分（仍保留"人脸至少 2/3 可见"的限制，由 `CropCalculator` 内部 `min_face_visibility=0.67` 控制；不满足时尝试缩小裁剪区域，若仍无法满足则回退到原图）
3. 如果指定了宽高比，严格按照宽高比裁剪
4. 检测失败或异常时直接返回原图（不进行任何缩放/裁剪）
5. 批次中尺寸不一致时，会以背景色居中填充对齐到批次内最大尺寸
6. 遮罩规则：
   - 有输入遮罩：与图像同裁剪/缩放（适应模式生成 letterbox 遮罩）
   - 无输入遮罩：输出全黑遮罩（与"内容区域"无关）

**注意：** 本节点采用"最大化人物主体显示"策略，会自动找到最大的裁剪区域，因此不提供边距参数。

### ZYF(按人物主体)比例缩放(2)节点

参数与"ZYF(按人物主体)比例缩放"节点相同，额外支持：

- **设备**：cpu/cuda（默认"cpu"，降低显存占用；推理结束后自动将模型移回 CPU 并清空 CUDA 缓存）
- **脸部模型**：选择人脸检测模型，默认 `face_yolov8m.pt`（缺失时显示推荐大小并自动下载）
- **人物模型**：选择人体检测/分割模型，默认 `person_yolov8m-seg.pt`（缺失时显示推荐大小并自动下载）

下拉列表中会显示模型文件名 + 估计大小（本地已存在的取实际文件大小，本地未存在但有提示值的取 `SIZE_HINTS` 中的参考值）。

### ZYF(聚焦人体裁剪)节点

**输入参数：**
- **图像**：输入图像
- **裁剪范围**：头像特写/胸部以上/腰部以上/臀部以上/膝盖以上/全身照（默认"臀部以上"）
- **裁剪比例**：智能裁剪比例，0.0=自动（使用裁剪范围预设），0.4-0.5=全身照，0.6-0.7=半身照，0.8-0.9=头像特写（默认0.0）
- **自动宽高比**：开启后根据图像横竖方向自动反转非"原始"的比例（默认关闭）
- **宽高比**：原始/自定义/1:1/3:2/4:3/16:9/2:3/3:4/9:16（默认"原始"）
- **自定义宽/高**：自定义宽高比的值（仅当宽高比选择"自定义"时生效）
- **缩放模式**：裁剪/拉伸/填充（默认"裁剪"）
  - `裁剪`：等比**放大**到能完整覆盖目标尺寸，再以**人脸中心**为锚点裁剪到目标尺寸（无人脸时退化为中心裁剪）
  - `拉伸`：等比缩放使整图能放入目标尺寸（不裁剪），周围用背景色 letterbox 填充
  - `填充`：直接拉伸到目标尺寸（不保持原比例）
- **采样方法**：lanczos/nearest/bilinear/bicubic（默认"lanczos"）
- **倍数取整**：8/16/32/64/128/256/512/无（默认"16"）
- **缩放到边**：无/最长边/最短边/宽度/高度/总像素(千像素)（默认"最长边"）
- **缩放到长度**：目标边的像素长度（默认1344）
- **背景颜色**：十六进制颜色代码（默认"#000000"）
- **人物索引**：选择第几个人物（默认0=所有人合成区域，1=最大人物，以此类推）
- **边距**：头部周围的额外边距（像素，默认50）
- **遮罩**（可选）：输入遮罩

**输出：**
- 图像：裁剪后的图像
- 遮罩：对应的遮罩

**裁剪策略：**
1. 优先使用头部检测模型精准定位头部（包含整个头部+头发）
2. 头部检测失败时自动回退到人脸检测
3. 以人脸为基准，根据人体标准比例计算各部位位置（下巴到胸部/腰部/臀部/膝盖/脚底）
4. **宽度按原图宽高比计算**：高度由裁剪范围决定，宽度按原图宽高比计算，裁剪掉所选范围周边多余的部分
5. 如果指定了宽高比（非"原始"），对裁剪范围后的图片进行对称比例裁剪（缩小）以达到目标比例；为保护人脸，目标尺寸不会小于人脸框的尺寸
6. 全身照优先使用身体检测框确定顶部和底部，再按原图宽高比计算宽度裁剪周边多余部分
7. 未检测到头部/人脸时返回原图
8. 遮罩规则：
   - 有输入遮罩：与图像同步裁剪/缩放
   - 无输入遮罩：输出全 1 遮罩（与图像同尺寸）

### ZYF(聚焦人体裁剪)预览节点

**输入参数：**
- **图像**：输入图像
- **裁剪范围**：头像特写/胸部以上/腰部以上/臀部以上/膝盖以上/全身照（默认"臀部以上"）
- **裁剪比例**：智能裁剪比例，0.0=自动（使用裁剪范围预设），范围 0.0–1.0、步长 0.05（默认0.0）
- **宽高比**：原始/自定义/1:1/3:2/4:3/16:9/2:3/3:4/9:16（默认"原始"）
- **自定义宽/高**：自定义宽高比的值（仅当宽高比选择"自定义"时生效；范围 1–32）
- **边距**：头部周围的额外边距（像素，范围 0–500、步长 10，默认50）

**输出：**
- 预览图像：带裁剪框和检测框标注的预览图像（与原图同尺寸，未经缩放/letterbox 处理）
- 裁剪信息：裁剪参数和检测信息的文本（每张图像一段，使用 `\n---\n` 分隔）

**预览节点不包含的参数：** 与正式裁剪节点相比，预览节点不显示缩放模式、采样方法、倍数取整、缩放到边、缩放到长度、背景颜色、人物索引和遮罩；预览仅展示裁剪范围/比例/宽高比/边距对裁剪框的影响。

### ZYF自动裁剪边框节点

**输入参数：**
- **图像**：输入图像
- **容差值**：颜色容差值（0-255，默认10）
- **最小裁剪尺寸**：最小裁剪尺寸（默认10）
- **遮罩**（可选）：输入遮罩

**输出：**
- 图像：裁剪后的图像
- 遮罩：对应的遮罩

## 技术细节

### 核心文件
- `nodes.py`：节点定义和主要逻辑
- `detection_manager.py`：人脸、头部和身体检测管理
- `crop_calculator.py`：裁剪区域计算
- `image_processor.py`：图像处理工具
- `models.py`：数据模型定义

### 依赖项
- PyTorch
- ultralytics（YOLOv8）
- Pillow
- NumPy
- requests

## 示例工作流

参考 `example_workflow.json` 文件，**当前随包发布的工作流仅包含**：
- 使用 `ZYFPersonCropAdvanced` 节点对图片进行智能裁剪并保存
- 使用 `ZYFAutoCropBorder` 节点裁剪纯色边框并保存

`ZYFHalfBodyCrop` 与 `ZYFHalfBodyCropPreview` 节点需自行添加到工作流中使用（README 中的截图示例可作为参考）。
